### グラフで「見える化」
- グラフであなたの「達成度」が確認できます。これにより、達成度のグラフ化によりあなたのモチベーションに火をつけます。

### 吹き出しで促される「目標設定」
- 目標を設定していない場合は、設定するように促す吹き出しが出現します。もう、目標を設定せざるを得ません。
